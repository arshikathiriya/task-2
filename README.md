# To-Do List Web App

## Overview
A simple To-Do List app built using HTML, CSS, and Vanilla JavaScript.

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks
- Instant updates without page reload

## Technologies Used
- HTML
- CSS
- JavaScript (ES6)

## How to Run
Open `index.html` in your browser or run with Live Server.

## Outcome
Learned DOM manipulation and event handling in JavaScript.
